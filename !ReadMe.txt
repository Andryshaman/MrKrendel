Описание проекта - что использовал.

У меня не было изначально конеретной цели. Если у других участников группы была необходимость разработать какой-то конкретный сайт (по работе и т.п.), то у меня такого не было.
Поэтому в качестве макета сайта я взял шаблон psd с сайта psd-html-css.ru Тематику придумал свою. Тексты, слоганы и пр. взял с готовых (живых рабочих) сайтов по теме.

В итоге что использовал из того что выучил:
1. Подключил файл normalize.css для поддержки кроссбраузерности.
2. При клике на номере телефона в хедере на мобильном устройстве должно вызываться приложение для звонков.
3. Подключил шрифты с г-фонтс.
4. Подключил шрифтовые иконки из библиотеки IcoMoon.
5. На кнопку "Оптовым клиентам" прикрутил форму обратной связи с помощью плагина Fancybox, для создания базы актуальных e-mail адресов с целью дальнейшей её продажи сторонним маркетологам (для организации рекламных рассылок:)
6. Сделал слайдер с использованием SkickSlider и стилизовал его.
7. Использовал флексы для распределения блоков в секции "Наши продукты" и др.
8. Добавил форму обратной связи в блоке "Связаться с нами" с инпутами, селектами и молоденькими поварихами:)
9. Добавил слайдер в блок "Новости".
10. Встроил в футер карту с г-мапс.
11. Постарался убрать весь повторяющийся код, объединить селекторы и свойства.

12. Добавил адаптивность под мобильные устройства с помощью медиа запросов. Их вынес в отдельный файл css.
	12.1 Свернул меню в кнопку с помощью JS.
	12.2 Все блоки сайта перестроил под маленькие экраны.
13. В меню добавил навигацию по странице.
